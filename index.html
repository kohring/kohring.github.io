Research: Multi-Agent Reinforcement Learning

My research focuses on computation of equilibria in markets and auctions via multi-agent reinforcement learning methods.

Since 2019

Financial Modelling using Online Machine Learning

The difficulty of predicting developments in the financial markets is to build a machine learning model, which can adapt as quickly as possible, while not neglecting learned dependencies. The project focused on an Online Deep Learning and on a Random Forest modeling approach.

Aug 2019

Master's Thesis: Statistical Analysis of Deep Learning Prediction Models using Bayesian Methods

Multiple deep learning approaches for solving regression problems while incorporating uncertainty were analyzed and modified in this work.
